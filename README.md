# Cypher-Flux 🛡️

A premium, real-time Security Operations Center (SOC) dashboard and threat detection system.

## 🚀 Quick Links
- [**System Architecture & Documentation**](./PROJECT_OVERVIEW.md)
- [Backend Documentation](./cypherflux-backend/README.md)
- [Frontend Documentation](./cypherflux-frontend/README.md)

## ✨ Core Features
- **Real-time Threat Monitoring**: Live traffic analysis and anomaly detection (Simulated).
- **Dynamic Flux Score**: Instant calculation of system security posture.
- **Glassmorphism UI**: High-fidelity, dark-themed professional dashboard.
- **Active Response**: One-click IP blocking and threat mitigation.

> [!NOTE]
> All data currently shown in the dashboard is generated by a heuristic simulation engine for demonstration purposes.


## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Socket.io.
- **Backend**: Python, Flask, SQLAlchemy, JWT, APScheduler.
- **Database**: PostgreSQL / SQLite.

## 📂 Repository Structure
- `cypherflux-backend/`: Python Flask API and detection engine.
- `cypherflux-frontend/`: React Vite application and dashboard UI.
- `docs/`: Technical specifications and configuration guides.
